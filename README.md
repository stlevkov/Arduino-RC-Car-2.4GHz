# Arduino-RemoteCar-2.4GHz
Arduino controlled remote car using 24L01 Transiever 2.4GHz

<h1> Hardware used in this project:</h1>

<ol>
  <li>Arduino Leonardo</li>
  <li>24l01 Tranciever</li>
  <li>X,Y, Joystick</li>
  <li>9V Battery</li>
</ol>
